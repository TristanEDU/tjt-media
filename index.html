TJT's Personal Media Server

Tired of Subscriptions?

I was too!

I was tired of getting a new subscription every time I found a new TV show I wanted to watch. It seemed like every time someone suggested a good audio book I needed to pay for it. Finally I said enough is enough. I built a server to store and host my media and anyone elses media who wants to share.

This server hosts TV shows, movies, audio books, music, and more. All you need is an account to access the media. If you have media you want to share, just reach out to me and we can get you set up with an account.
